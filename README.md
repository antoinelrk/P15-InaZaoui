# Ina Zaoui

## Présentation

Le site Ina Zaoui est une plateforme professionnelle dédiée à la présentation du parcours, des créations et des activités de l’artiste Ina Zaoui. Elle met en valeur son univers visuel et artistique à travers une galerie soignée, des contenus éditoriaux, ainsi qu’un espace d’actualités permettant de suivre ses expositions, projets et collaborations. L’objectif du site est de proposer une vitrine élégante, moderne et accessible, reflétant fidèlement l’identité de l’artiste et facilitant la prise de contact.

## Prérequis

- Docker (Dernière version recommandée)
- Le package unix `make`

**Pour utiliser make sous windows ...**

Télécharger le binaire de GnuWin32 Make (make.exe).
Vous pouvez récupérer un exécutable compatible depuis les distributions GnuWin32 ou via des dépôts tiers précompilés.

Placer make.exe dans un dossier, par exemple :

`C:\Program Files\GnuWin32\bin`

**Ajouter ce dossier au PATH Windows :**

Panneau de configuration → Système → Paramètres système avancés
Variables d’environnement → Modifier Path
Ajouter le chemin contenant make.exe
